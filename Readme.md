# aggressive-tokenizer

  Aggressive tokenizer (from node-natural)

## Installation

    $ component install danzajdband/aggressive-tokenizer

## Usage

```js

var tokenizer = require('aggressive-tokenizer');

tokenizer.tokenize('viva la pepa    ');
  //['viva', 'la', 'pepa']

```
